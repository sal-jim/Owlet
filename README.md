## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Start the server:

```bash
pnpm run owlet
```

3. Open [http://localhost:4173](http://localhost:4173) in your browser.

> Set `OWLET_PORT` to use a different port.

## Development

For live-reload during development:

```bash
pnpm run owlet:dev
```
